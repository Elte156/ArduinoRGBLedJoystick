# Arudino RGB LED Joystick Project

Moving the joystick in specific directions will blend and create different colors on your LED.

#### Hardware needed for this project
* Arduino
* RGB LED
* Joystick



#### Pin Config
* Arduino
	* Pin 2 => S-K
	* Pin 11 => Red
	* Pin 10 => Green
	* Pin 9 => Blue
* RGB (pins left to right)
	* Red
	* Ground (longest pin)
	* Green
	* Blue
* Joystick
	* GND
	* VCC
	* S-Y
	* S-X
	* S-K
